Drive builder¶

Introduction¶

Drive Builder is a software package that allows the user to create, edit and save a set of bridge boards. The boards are saved in a file which can be used, for example, to interact with the i-Dealer board dealing machine or by a teacher to save a set of hands for lessons.

The user has control over the number of boards dealt in a drive and can select random boards or boards generated according to selected criterea.

Users¶

The system comes with a default user. If the sytem will be used by more than one user, it might be sensible to set up different users. The system stores, against each user, the default path to the drive files, the criterea used to generate drives and whether or not to confirm exit if changes have been made to a drive.

New drives¶

To generate a new drive click on the ‘Generate’ button or use Menu>File>New. A drive will be generated using the criterea selected by the user.

Saving, opening and deleting drives¶

Opening a saved drive¶

Click on Menu>File>Open. A dialog will open allowing you to select the drive.

Saving a drive¶

Click on Menu>File>Save. A ‘Save’ dialog will appear with a default name for the drive based on the date and time of creation. Change this name if required to something more meaningful. Drives are saved with the extension ‘drv.

Save an existing drive under a different name¶

Click on Menu>File>SaveAs. A ‘Save as’ dialog will appear. Enter the new name for the drive.

Deleting drives¶

The system does not hae a facility to delete drives. Should you wish to delete a drive, locate the drive file and use your system’s normal method for deleting files.

Drive information¶

Each board in the drive is listed in order with a default description. The remaining columns show:

shapes: the shape of each of the four hands on the board in the order N, E, S, W.

max length: the longest suit in any hand on the board.

points: the high card points in each hand.

max points: the high card points in the strongest hand on the board.

Board manipulation¶

Note changes made to a drive are not saved to file unless you explicitely ‘Save’ the drive (Menu>File>Save).

Editing a boards in a drive¶

The cards in each hand of a board can be changed by using the board editor. To enter the board editor double click on the board that you wish to edit in the board list, right click on the mouse and select ‘edit’ in the context menu.

Changing the order of boards in a drive¶

Select the board that you wish to move in the board list. Right click on the mouse and a menu will appear; select ‘move up’ or ‘move down’ as appropriate.

Deleting a board from a drive¶

Select the board that you wish to delete in the board list; ight click on the mouse and a menu will appear; select ‘delete’; the board will be removed.
